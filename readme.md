This application serves for quick fishes split based on the entry stack for poker cash game, it is useful to menage tables, take track of records and keep in memory owed cash at end of the games.

## Technologies

This application was built using:

- Angular 21
- Ngx-Bootstrap
- Go 1.21
- MySql

## Responsivness

The responsivness is done using the Ngx-Bootstrap library

## Backend

The backend is done using Go and MySql as database
